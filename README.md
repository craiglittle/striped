# Striped

A clean implementation of Stripe API Ruby bindings.


## Installation
```ruby
gem install striped
```


## Supported Ruby interpreters

This gem officially supports and is tested against the following Ruby interpreters:

* MRI 1.9.2
* MRI 1.9.3
* MRI 2.0.0
* JRuby in 1.9 mode
* Rubinius in 1.9 mode
